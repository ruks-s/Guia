HTML - O que é? E o que será visto abaixo?

O que é?

É basicamente uma linguagem de marcação (detalhe que não é linguagem de programação) utilizada para estrutura, organizar e criar conteúdo para uma página - a grosso modo, é a parte visível escrita no site, o que será alvo posteriormente de CSS (estilização) e de JS (interações na página).

O que será visto abaixo?

Abaixo se terá um panaroma bem geral acerca de conceitos, elementos e atributos presentes dentro do HTML - não o tornará um expert em HTML, mas servirá como uma boa base independentemente do seu objetivo nessa área (front-end).

CSS - O que é? E o que será visto abaixo?

O que é?

É a linguagem utilizada para estilização do HTML, ou seja, é o que define a aparência de uma webpage, visto que depender apenas da estilização padrão de um navegador é algo bem limitado - a grosso modo, é a skin do site.

O que será visto abaixo?

Diferentemente de HTML, CSS é algo bem mais conceitual do que prático (apesar de exigir muita prática como qualquer ação relacionada a programação), visto que se tem MUITAS propriedades, torna-se mais fácil apenas ter uma ideia geral acerca de como funciona sistemas, e saber o que e onde procurar certas propriedades a depender do seu interesse - logo, o foco da página abaixo, se torna isso, formar uma base nos principais conteúdos que temos que saber como DEVs (como Grid e Flexbox), além de conhecimentos básicos estruturais, etc.
